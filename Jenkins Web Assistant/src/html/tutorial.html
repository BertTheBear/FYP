Explanation

Jenkins Web Assistant is an Extension for chrome aimed at aiding web usage. The application is intended to be able to provide a scheduling system that will prompt users at certain times with site recommendations, and will also provide recommendations for similar sites to those being browsed. All capabilities of the extension are fully customisable and may be switched on or off in the Advanced Options Page.

A brief tutorial for the major functions is given below.

Scheduler

The scheduler is the mechanism by which the extension will prompt the user at certain times with certain recommendations. A sample schedule is given below.

Entries are divided into User Entries and Automatic Entries.

  • User entries are in green.
    • These are created by the user using a form like that below. This is done on the Interface Page.
    • These entries can be removed by clicking on the X on the right-hand side of the schedule entry.
    • Removing an entry will prompt a notification window asking if you would like to undo the operation, in case you make a mistake.
    • These will prompt notifications that can be clicked to open that website in a new or existing tab.
    • Notifications can be turned on and off in the Options page, and may be temporarily turned off by selecting the "Prevent notifications this session" option in the popup window by clicking the icon in the extension bar at the top of the screen.
    • Click here for a sample User notification
  • Automatic entries are in blue.
    • These are created automatically by the extension if the option is selected. This can be selected on the Options Page
    • This is done when the web browser is opened and if the Process History at the top of the Interface Page button is pressed.
    • These entries can be removed by clicking on the X on the right-hand side of the schedule entry similar to User entries.
    • Clicking the Accept button will accept the recommendation and change the colour to green. This will show a notification upon successful validation of the schedule item.
    • Notifications for automatic schedule entries are turned off by default. These will need to be turned on using the Advanced Options Page or the Interface Page

Recommendations

Another capability of the Extension is the Recommender. This function provides you with example websites based upon the site you are currently viewing. This can be seen by clicking on the icon in the extension bar at the top of the screen. This will process the current web page and assign it to a category. The extension will then select a similar website in the same category and show it to you. Clicking on these recommendations will open the website in a new tab. Not all websites will have recommendations.

These recommendations can be accepted or rejected using the Accept and X buttons. Accepted and rejected recommendations can also be viewed on the Recommendations Feedback Page
